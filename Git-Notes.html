(04) Checking Status
--------------------
4.1 $ git status

(05) Checking Differences
-------------------------
5.1 $ git diff

5.3 $ git diff --staged

(06) Unstared & Undoing Changes
--------------------
6.1.1 $ git reset (commit_id)

6.1.3 $ git reset .

6.2.1 $ git checkout (file name)
6.2.2 $ git checkout .

(07) Pull & Push
----------------
7.1 $ git push -u origin (branch_name)
7.2 $ git push

7.3 $ git pull origin (branch_name)
7.4 $ git pull


(09) Viewing Commit History
---------------------------
9.1 $ git log
9.2 $ git log --oneline
9.3 $ git log -p -(number of commit)


(10) Cloning and Creating Copies
--------------------------------
10.1 $ git clone (repository_url)
10.2 $ git clone (repository_url) .
10.2 $ git clone (repository_url) (folder_name or Folder_path)
10.3 $ git clone (branch_name) (repository_url)


(11) Branching and Merging
--------------------------
11.1 $ git branch {List all branches in your repository}
11.2 $ git branch (branch_name) {Create a new branch}
11.3 $ git checkout (branch_name) {Switch to a different branch}
11.4 $ git merge (branch_name) {Switch to a different branch}
11.5 $ git branch -d (branch_name) {Delete a branch}

11.6 $ git push origin --delete (branch_name)


(12) Remote Repository Interaction
----------------------------------
12.1 $ git remote
12.2 $ git remote -v {List all remote repositories.}
12.3 $ git remote add (remote_name) (repository_url)  {Add a new remote repository}
12.3 $ git remote remove (remote_name) {Remove a remote repository}
12.3 $ git remote rename (old_name) (new_name) {Rename a remote}
12.3 $ git fetch (remote_name) {Fetch changes from a specific remote}
